# ``SwiftDisc``

A Discord API wrapper for Swift with typed models, actor-based concurrency, and automatic gateway management.

@Metadata {
  @PageImage(purpose: icon, source: swiftdisc-logo)
  @PageColor(blue)

SwiftDisc brings together the full Discord REST API, Gateway events, typed models, and high-level bot-building tools — all built on Swift 6.2's async/await and actor-safe concurrency from day one.
